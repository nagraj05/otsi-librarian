## вңҰ What is OTSI Librarian?

OTSI Librarian is a lightweight office book-lending tracker. Anyone on the team can open the home page and immediately see which books are checked out and who has them. The admin (you) logs in to add new borrow records, mark books as returned, or delete entries.

When you add a book, the title is looked up via the Google Books API and all metadata вҖ” cover art, authors, description, publisher, page count, categories, ISBN, and rating вҖ” is fetched and stored automatically.

## вңҰ How It Works

```
Admin types a book title
  в””в”Җв–ә /api/books/search?q=...
        в””в”Җв–ә Google Books API вҶ’ volumeInfo (cover, authors, pages, ISBN, ratingвҖҰ)
              в””в”Җв–ә BookSearch dropdown вҶ’ user picks a result
                    в””в”Җв–ә addBorrow() server action
                          в””в”Җв–ә INSERT INTO borrows (all metadata stored once)
                                в””в”Җв–ә revalidatePath('/') + revalidatePath('/admin')

Public visitor opens /
  в””в”Җв–ә getBorrows() вҶ’ SELECT * FROM borrows ORDER BY borrowed_at DESC
        в””в”Җв–ә reduce into Record<borrowerName, Borrow[]>
              в””в”Җв–ә PersonCard grid (currently out) + ReturnedCard grid
```

Book detail pages read entirely from the **database** вҖ” no second API call needed because all metadata was captured at borrow time.
